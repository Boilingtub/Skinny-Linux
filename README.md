# Skinny-Linux
Skinny Linux is a very small unprofesional linux disto made around myself ! The aim is to make a linux desktop that is as small as possible while still being useble to me.
It is Based on Debian 11 (Bullseye):

To download graphics environment install script
curl -LJO https://github.com/Boilingtub/Skinny-Linux/raw/main/Skinny-Desktop-Setup.sh

Give the script executable premissions
chmod +x Skinny-Desktop-Setup.sh

Run Setup Script
./Skinny-Desktop-Setup.sh
