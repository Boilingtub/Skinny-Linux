# Skinny-Linux
<p align="center">Skinny Linux is a very small unprofesional linux disto made around myself ! The aim is to make a linux desktop that is as small as possible while still being useble to me.</p>
<p align="center">t is Based on Debian 11 (Bullseye):</p>

<p align="center">To download graphics environment install script</p>
```
<p align="center"> <b>curl -LJO https://github.com/Boilingtub/Skinny-Linux/raw/main/Skinny-Desktop-Setup.sh </b></p>
```

<p align="center">Give the script executable premissions</p>
```
<p align="center"> <b> chmod +x Skinny-Desktop-Setup.sh </b></p>
```

<p align="center">Run Setup Script</p>
```
<p align="center"> <b>./Skinny-Desktop-Setup.sh </b></p>
```
