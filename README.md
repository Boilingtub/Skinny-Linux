# Skinny-Linux
<p align="center">Skinny Linux is a very small unprofesional linux disto made around myself ! The aim is to make a linux desktop that is as small as possible while still being useble to me. It is Based on Debian 11 (Bullseye)</p>

<p align="center">To download graphics environment install script:</p>

```
curl -LJO https://github.com/Boilingtub/Skinny-Linux/Install/raw/main/Install/Skinny-Desktop-Setup.sh
```

<p align="center">Give the script executable premissions:</p>

```
chmod +x Skinny-Desktop-Setup.sh
```

<p align="center">Run Setup Script:</p>

```
./Skinny-Desktop-Setup.sh
```
